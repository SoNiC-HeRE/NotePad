# NotePad
An interactive web Notepad to keep track of your notes on the Go!

## Specs
Languages used:
<ul>
<li>HTML5</li>
<li>CSS3</li>
<li>JavaScript</li>
</ul>

Tools,Frameworks and Libraries Used:
<ul>
<li>ReactJS</li>
<li>NodeJS</li>
<li>VS Code</li>
<li>Netlify</li>
<li>Git Bash</li>
</ul>

Preview: notepad-sonic.netlify.app

## Install
```bash
npm install
```

## Run
```bash
npm start
```
